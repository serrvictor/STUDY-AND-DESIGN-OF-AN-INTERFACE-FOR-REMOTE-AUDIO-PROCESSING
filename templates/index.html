For PC Users: Ctrl + F5 to update plots

For Phone Users: you'll have to manually clear any cache data associated with the site and reload the page to update plots

For Chrome Phone Users:

  1. Tap on the three-dot menu icon to open settings.
  2. Under Advanced, open site settings and go to Data stored.
  3. Tap on the site URL you wish to hard refresh and tap on the Clear & reset button.
  4. Return to the website and use the tree-dot menu icon and tap on the refresh button.
